# 🎧 Jamming Room - Real-Time Music Sync App

## 📌 Project Overview
Jamming Room is a real-time collaborative music listening application where multiple users can join a room and listen to music together in sync. The host controls playback, and all participants experience synchronized audio.

---

## 🚀 Features

### 🔹 Basic Features
- Create Room
- Join Room
- Play Music via URL (MP3)
- Real-Time Music Sync
- Host Control (Play/Pause/Change Track)

### 🔹 Optional Features
- Chat System 💬
- User Login 🔐
- Playlist Support 🎶

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- HTML5, CSS3
- Axios

### Backend:
- Node.js
- Express.js
- Socket.IO (Real-time communication)
